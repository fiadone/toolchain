# @fiad/bundler

A [webpack](https://webpack.js.org/) based bundler featuring [babel](https://babeljs.io/), [sass/scss](https://sass-lang.com/) support, and [postcss](https://postcss.org/)

---

## Get started

```
cd my-project
npx @fiad/bundler
```

## Commands

#### Development
```
npm run dev
```

#### Production build
```
npm run build
```
